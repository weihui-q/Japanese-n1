# N1词汇导入指南

## 方法1: 手动录入(推荐用于少量词汇)

### 步骤:
1. 打开PDF文件,查看词汇列表
2. 在应用中点击左侧边栏的"インポート"(导入)
3. 选择"単語"类型
4. 按照以下JSON格式输入词汇数据
5. 点击"インポート"按钮

### JSON格式示例:

```json
[
  {
    "id": "w011",
    "kanji": "相応しい",
    "kana": "ふさわしい",
    "romaji": "fusawashii",
    "meaning": "适合的,相配的",
    "example": "この服装はパーティーに相応しい。",
    "exampleTranslation": "这套服装适合派对。",
    "category": "形容詞",
    "difficulty": 3,
    "tags": ["評価", "適合"]
  },
  {
    "id": "w012",
    "kanji": "惹きつける",
    "kana": "ひきつける",
    "romaji": "hikitsukeru",
    "meaning": "吸引,吸引住",
    "example": "彼の演説は聴衆を惹きつけた。",
    "exampleTranslation": "他的演讲吸引了听众。",
    "category": "動詞",
    "difficulty": 4,
    "tags": ["魅力", "影響"]
  }
]
```

## 方法2: 批量导入(推荐用于大量词汇)

### 步骤:
1. 创建一个CSV或Excel文件,包含以下列:
   - id (编号,如 w011, w012...)
   - kanji (汉字)
   - kana (假名)
   - romaji (罗马字)
   - meaning (中文意思)
   - example (日语例句)
   - exampleTranslation (例句翻译)
   - category (词性分类)
   - difficulty (难度 1-5)
   - tags (标签,用逗号分隔)

2. 使用在线工具或脚本将CSV转换为JSON格式

3. 在应用的"インポート"页面导入JSON数据

## 方法3: 使用Python脚本转换(如果你有编程基础)

### Python脚本示例:

```python
import json
import pandas as pd

# 读取Excel/CSV
df = pd.read_csv('n1_words.csv')

# 转换为所需格式
words = []
for _, row in df.iterrows():
    words.append({
        "id": row['id'],
        "kanji": row['kanji'],
        "kana": row['kana'],
        "romaji": row['romaji'],
        "meaning": row['meaning'],
        "example": row['example'],
        "exampleTranslation": row['exampleTranslation'],
        "category": row['category'],
        "difficulty": int(row['difficulty']),
        "tags": [tag.strip() for tag in row['tags'].split(',')]
    })

# 保存为JSON
with open('n1_words.json', 'w', encoding='utf-8') as f:
    json.dump(words, f, ensure_ascii=False, indent=2)

print(f"已转换 {len(words)} 个词汇")
```

## 词性分类参考:

- 名詞 (名词)
- 動詞 (动词)
- 形容詞 (形容词 - い形容词)
- 形容動詞 (形容动词 - な形容词)
- 副詞 (副词)
- 連体詞 (连体词)
- 接続詞 (连词)
- 感動詞 (感叹词)
- 名詞・他動詞 (名词/他动词)
- 名詞・自動詞 (名词/自动词)

## 标签建议:

根据词汇的使用场景添加标签,例如:
- 社会, 経済, 政治, 文化, 教育, 技術, 環境, 人間関係, 感情, 思考, 時間, 空間, 数量, 評価, 変化, 状態, 動作...

## ID编号规则:

- 单词: w001, w002, w003... (现有10个,从w011开始)
- 语法: g001, g002, g003... (现有8个)

## 注意事项:

1. **ID必须唯一** - 不要与现有词汇重复
2. **编码使用UTF-8** - 确保日语字符正确显示
3. **必填字段** - id, kanji, kana, meaning 是必需的
4. **难度等级** - 1(最简单)到5(最难)
5. **例句可选** - 但强烈建议添加,有助于理解
