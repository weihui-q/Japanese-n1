# JLPT N1 学习网站 - 启动说明

## 快速启动

### 方法1: 使用批处理文件(推荐)
双击项目文件夹中的 `start.bat` 文件即可启动。

### 方法2: 使用命令行
```bash
# 1. 打开PowerShell或命令提示符
# 2. 进入项目目录
cd "c:\Users\user17\Desktop\新建文件夹\日语\git words\jlpt-n1-study-new"

# 3. 启动开发服务器
npm run dev
```

### 方法3: 使用npx
```bash
cd "c:\Users\user17\Desktop\新建文件夹\日语\git words\jlpt-n1-study-new"
npx vite
```

## 访问应用

启动成功后,在浏览器中打开:
- **http://localhost:5173**

如果端口被占用,Vite会自动使用其他端口,请查看命令行输出的URL。

## 如果遇到问题

### 1. 依赖未安装
```bash
npm install
```

### 2. 端口被占用
```bash
npx vite --port 3000
```

### 3. 清除缓存重新启动
```bash
# 删除node_modules和package-lock.json
Remove-Item -Recurse -Force node_modules
Remove-Item package-lock.json

# 重新安装
npm install

# 启动
npm run dev
```

## 功能说明

启动后你可以:

1. **首页** - 查看学习统计和快速入口
2. **単語** - 浏览和搜索N1单词
3. **文法** - 浏览和搜索N1语法  
4. **フラッシュカード** - 使用闪卡记忆系统学习
5. **テスト** - 做选择题测试
6. **進捗** - 查看学习进度
7. **インポート** - 导入更多单词和语法数据

## 示例数据

项目已包含:
- 10个N1单词示例
- 8个N1语法示例

你可以在"インポート"页面导入更多数据。

## 技术栈

- React 18 + TypeScript
- Vite (构建工具)
- TailwindCSS (样式)
- Zustand (状态管理)
- Framer Motion (动画)
- IndexedDB (本地存储)
